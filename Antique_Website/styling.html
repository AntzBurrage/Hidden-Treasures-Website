Use of Colours

Using a color scheme web application called COOLORS the following colour scheme was chosen. During research it was found that the majority of Antique Websites went for the old fashioned and dated themes. The Logic of colour use within "Hidden Treasures" was to try for a more modern, lively but not tacky aesthetic. The Following were the Colours Choose as noted above and where they were implemented.

  • #C9CDD9(Languid Lavender) - This was used as the Body colour.

  • #454564(Independence) - This was used as the Navigation and Footer base colour.

  • #333131(Jet) - This was used as base colour for all Links, Captions, Paragraphs and Headings within the Body.

  • #F6F6F6(White Smoke) - This was used as the Navigation and Footer Anchors and Links.

  • #B1A275(Pale Taupe) - This was used to indicate that a Link is currently active, backgrounds and some buttons.

  • #CEB210(Lemon Curry) - This was used to indicate that a Link is currently active

There isn’t any concern for legibility using this colour scheme as the correct use of contrast and layout will ensure users with Deuteranopia, Protanopia, Tritanopia or other conditions can navigate and enjoy their visit to "Hidden Treasures".
